## git add = working directory-> stagin area

git add : 파일명.js

git add . : 현재 디렉토리 안 모든파일 추가

git status :  파일변경등의 파일상태 확인

<br>

## 깃 저장소 반영
## git commit -m = staging area -> git repository

git commit -m "커밋 메세지"

git commit --amend
- 앞 커밋 메세지 내용 변경

저장소 반영 내역 확인
git log
